# Claude Code Project Configuration

Để biết hướng dẫn phát triển đầy đủ dành cho agent, xem [AGENTS.md](./AGENTS.md).

Tất cả Claude agents làm việc trên dự án này cần tuân theo các hướng dẫn phát triển trong AGENTS.md, bao gồm:
- Cấu trúc dự án và quy ước thư mục
- Lệnh và quy trình làm việc chính
- Kiến trúc và các pattern
- Các quy tắc và tiêu chuẩn phát triển
